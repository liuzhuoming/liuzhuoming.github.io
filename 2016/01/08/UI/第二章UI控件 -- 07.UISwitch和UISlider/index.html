2.了解并使用UISlider

在iOS中经常可以看到类似于进度条的控件,比如,音量的控制、播放进度的控制。在 iOS 开发中,这种使用滑块来改变数值的控件称为滑块控件。滑块控件使用 UISider 表示,它继承自 UIControl,是一个可以与用户进行交互的控件。

UISlider常用的属性表

常用属性 定义
value 当前的值
minimumValue 当值可以改变时，滑块可以滑动到最小位置的值，默认为0.0
maximumValue 当值可以改变时，滑块可以滑动到最大位置的值，默认为1.0
continuous 如果设置YES，在拖动滑块的任何时候，滑块的值都会改变。默认设置为YES
minimumValueImage 滑块条最小值处设置的图片，默认为nil
maximumValueImage 滑块条最大值处设置的图片,默认为nil
thumbTintColor 当前滑块的颜色，默认为白色
currentThumbImage 当前滑块的图片
