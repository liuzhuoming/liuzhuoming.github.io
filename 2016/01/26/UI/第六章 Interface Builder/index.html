第六章 Interface Builder

一、学习目标

  • Xib和StoryBoard介绍使用
  • Xib创建UIViewController
  • 使用StoreBoard创建多UIViewController
  • UIViewController中View的创建

二、课程内容

1.Xib和StoryBoard介绍

xib的基础知识

  • .xib ＝ nib；
  • .xib的加载：

[[NSBundle mainBundle] loadNibName:@"rowView" ower:nil options:nil];

xib 和storyboard 的不同点

  • storyboard：描述软件界面，大范围，重量级，比较适合描述整个软件的所有界面；
  • xib：描述软件界面，小范围，轻量级，比较适合描述小界面（局部界面）；

xib 和storyboard 的相同点

本质都是转成代码

File‘s owner的使用

  • 在xib文件中设置file‘s owner的class属性
  • 建立File‘s owner 与控件之间的联系
  • 利用代码加载xib，传递owner参数

2.使用Xib创建UIViewController

  • 创建一个ViewController 并勾上同时创建一个xib的选项

  • 这时新增的代码文件除了我们之前熟悉的.h和.m文件还多了一个.xib文件

  • 从右下角拖拽控件到中间的View上,并在右上角调整各类控件的样式

  • 通过IBAction关联点击事件

  • 通过Outlet关联属性

思考为什么是weak，如果改成strong有什么不同？

  • 实例化XibVC，并设置成根视图。
